# Sanya Boilerplate

## Перед запуском
Создайте файл .env по образцу .env.example


## Версии пакетов
- React 18
- redux 4
- React-redux 8
- Redux toolkit 1.9
- Dotenv 16
- React-dnd 16
- React-icons 4
- Styled-components 5

## Команды запуска

- `npm install`
- `npm start` - запустить для разработки
